自我要求

明于世，严于己，厚于人。

明于世 — 对世界保持清醒的认知

看清现实的复杂与残酷，不天真、不自欺、不逃避真相。既包括对外部世界的洞察（人性、规律、利弊），也包括对自身处境的清醒认识。”明”不是冷漠，而是看得真切。

严于己 — 对自己保持高标准的要求

在无人监督时仍然自律，在利益受损时仍然诚实，在犯错时直面而非推卸。”严”不是自我折磨，而是不放低对自己的期待。

厚于人 — 对他人保持宽厚与善意

理解他人处境，给予真实的信任与善意，不以苛刻的标准评判他人，愿意付出而不斤斤计较。”厚”不是软弱，而是一种有底气的慷慨。

一、认知与思维

  1. 对真相的承受度——能接受不利于自己的事实，不因现实令人不适便回避或扭曲它
  2. 明辨是非的能力 ——区分事实与观点、本质与表象、重要与紧急
  3. 批判性思考 ——不盲从权威，对任何结论追问前提与逻辑
  4. 换位思考 ——真正进入他人处境，而非仅以自身逻辑推测他人
  5. 全面思考 ——主动寻找反例与盲点，避免只看支持自己观点的证据
  6. 独立判断 ——大量吸收信息后，经过自己的消化与过滤，形成独立结论，而非人云亦云

二、信息与学习

  1. 持续、主动地获取信息 ——广泛涉猎，不局限于舒适区的领域
  2. 大量阅读与深度学习 ——区分消遣性阅读与结构性学习，两者并重
  3. 知识转化为实践 ——所学必须经过实践检验，否则只是信息堆积
  4. 从错误中系统性学习 ——不只是”吸取教训”，而是分析根因、建立机制防止重复
  5. 善于听取合理建议 ——无论建议者身份高低，只看建议本身是否正确

三、分析与解决问题

  1. 结构化分析问题的能力 ——将复杂问题拆解为可处理的部分，找到核心矛盾
  2. 在不完整信息下做决策的能力 ——现实中永远没有完美信息，能在模糊中果断行动
  3. 预判后果与风险 ——行动前评估可能的结果，包括最坏的情形
  4. 大局观 ——不因局部得失而失去对整体方向的把握
  5. 目光长远 ——能为长期利益放弃短期诱惑，不被眼前的压力压缩视野

四、执行与行动

  1. 极致的执行力 ——决定了就做，做了就做到位，不拖延，不敷衍
  2. 理性决策 ——基于证据与逻辑做判断，而非情绪、习惯或从众
  3. 落子无悔 ——决策后全力投入，不内耗于反复后悔；但对新信息保持开放，必要时修正
  4. 敢于担责 ——出了问题不推卸，不找借口，直接面对后果
  5. 不怕犯错 ——将犯错视为必然的学习成本，而非对自我价值的否定

五、心理素质

  1. 任何情况下保持冷静 ——情绪不驾驭判断；情绪可以存在，但不能主导决策
  2. 承压能力 ——在高压、逆境、不确定性中维持正常的判断与行动力
  3. 对真相的心理承受力 ——不因现实残酷便转向自我欺骗或逃避
  4. 延迟满足 ——能克制即时诱惑，为更重要的目标积累资源与耐心
  5. 不依赖外部认可 ——自我评价有独立的内在标准，不随他人的褒贬而剧烈波动

六、品格与原则

  1. 礼貌而不谄媚 ——对任何人保持基本尊重，但不为讨好而迎合
  2. 自信而不自傲 ——清楚自己的能力边界，既不妄自菲薄，也不轻视他人
  3. 谦虚而不自卑 ——承认自己的不足，但不因此否定自身的价值
  4. 诚实 ——对他人不说谎，对自己不自欺，包括在利益受损时仍然如此
  5. 敢于批评 ——在该指出问题时不沉默，即使对象是权威或强者
  6. 敢于拒绝 ——能对不合理的要求说”不”，不因恐惧人际冲突而妥协原则
  7. 言行一致 ——所说与所做之间没有长期的落差

七、自我迭代

  1. 主动发现自身缺点 ——不等别人批评，自己建立定期审视自我的习惯
  2. 缺点→改进→验证→循环 ——改进是一个持续的闭环，而非一次性的自我感动
  3. 区分”可改变”与”可接受”的局限 ——有些弱点值得克服，有些特质值得接纳，需要智慧区分

八、与他人的关系

  1. 理解他人的真实处境与动机 ——不以表面行为下判断，努力理解背后的逻辑
  2. 建立并维护真实的信任 ——信任来自长期一致的行为，而非语言上的表态
  3. 处理真实利益冲突的能力 ——在自身利益与他人利益产生冲突时，能公正、透明地处理
  4. 合作而非零和 ——在可能的情况下寻找双赢，而非默认关系是竞争性的
  5. 给予他人真实的、有建设性的反馈 ——不只说好话，也不只批评，而是对对方真正有用

九、对世界的态度

  1. 对复杂性保持尊重 ——不简化不该简化的问题，能与不确定性和模糊性共存
  2. 独立于时代情绪 ——不被集体性的焦虑、狂热或悲观所裹挟
  3. 对自身局限保持清醒 ——知道自己的认知、经验、视野都有边界，并将此内化为行动准则
