> 「拾叶」者，拾取散落之思绪也。这是一座**无需构建框架**的个人笔记书斋：每篇笔记就是一个普通的 Markdown 文件，放进 `notes/` 文件夹即可。

## 它能做什么

- **模糊搜索**：左侧搜索框支持不连续匹配，比如输入 `js闭包` 或 `dk命令` 也能找到对应文章；标题、标签、正文全部参与检索。按 `/` 键可快速聚焦搜索框。
- **搜索语法**（Google 式）：

| 写法 | 含义 |
|---|---|
| `闭包 面试` | 同时包含两个词（AND） |
| `"缓存穿透"` | 精确短语 |
| `docker -速查` | 排除含「速查」的结果 |
| `闭包 OR 缓存` | 任一命中即可 |
| `tag:读书` `title:闭包` | 按标签 / 标题过滤 |
| `before:2026-04` `after:2026-01` | 按日期过滤 |

普通词依然支持适度模糊（`js闭包`、`dk命令` 能命中），但限制了字符间隙，不会再"什么都搜得到"。
- **三态主题**：右上角按钮在「跟随系统 → 浅色 → 深色」之间循环切换。
- **朱批之笔**：右下角的毛笔是这座书斋的灵魂。古人读书以朱笔圈点，你也可以——点开它，选一支颜色，在文章上随意**圈圈划划**；荧光黄适合划重点，「一键清屏」即可还原素纸。

## 怎么添加一篇新笔记

**第一步**，在 `notes/` 文件夹里新建一个 `.md` 文件，开头写三行信息（front matter）：

```markdown
---
title: 我的新笔记
date: 2026-06-15
tags: 随笔, 生活
---

正文从这里开始，写普通的 Markdown 即可……
```

文件名随意，推荐 `2026-06-15-my-note.md` 这样的格式。如果懒得写 front matter 也没关系：标题会取正文第一个 `#` 标题，日期会从文件名里猜。

**第二步**，在项目根目录运行一次索引脚本：

```bash
python3 build.py
```

它会扫描 `notes/` 并更新 `notes/manifest.json`。刷新页面，新笔记就出现在目录里了。

> 如果把仓库托管在 GitHub 上，项目里附带的 GitHub Action 会在每次 push 后**自动**重建索引——也就是说，你只需要提交 .md 文件，其他什么都不用管。

## 书斋的其他房间

- **代码集**：把自己写的小脚本丢进 `snippets/`（.py / .sh / .js …），文件开头用一行注释 `#: 描述` 写说明、`# tags: a, b` 打标签，运行 `build.py` 后就会出现在「代码集」页面，带语法高亮和一键复制。
- **网站收藏 / 工具箱**：编辑 `collections/websites.json` 与 `collections/tools.json`，每条收藏是 `{ name, url, icon, desc, tags }`。
- **提示词收藏**：编辑 `collections/prompts.json`，每条是 `{ title, desc, tags, prompt }`，页面带标签筛选、一键复制与长文折叠。
- **Obsidian 适配**：`[[双链]]`、`==高亮==`、`> [!tip]` 标注框、`![[图片]]`（附件放 `assets/`）、`%%注释%%`、任务清单都能正确渲染；front matter 容忍全角冒号与 Windows BOM。见 [[Obsidian 语法演示：这座书斋认识双链]]。
- **关于页**：编辑 `pages/about.md`。

## 开屏的一图一句

每次打开网站，会随机抽一张背景配一句诗。配置在 `collections/splash.json`：

- `quotes` 里加自己喜欢的句子（`text` + `by`）；
- `backgrounds` 默认是 6 个内置场景（星汉 / 暮霭 / 山岚 / 沧海 / 烬火 / 月白），想用自己的照片，把图放进 `assets/splash/`，再加一条：

```json
{ "type": "image", "src": "assets/splash/my-photo.jpg", "fallback": "muai", "light": false }
```

`light: true` 表示这是浅色图片，文字会自动换成墨色。`"enabled": false` 可整体关闭开屏。深链直达某篇文章时不会弹出开屏。

## 本地预览与部署

浏览器出于安全限制不允许网页直接读取本地文件，所以**双击打开 index.html 是看不到笔记的**。本地预览请在项目目录运行：

```bash
python3 -m http.server 8000
# 然后访问 http://localhost:8000
```

部署则把整个文件夹放到任何静态托管（GitHub Pages、Vercel、Cloudflare Pages）即可。
